# Melodic.SharedKernel.Analyzers

Roslyn analyzers for enforcing Medley architecture patterns and module boundaries.

## Installation

```shell
dotnet add package Melodic.SharedKernel.Analyzers
```

Or add to your project file:

```xml
<PackageReference Include="Melodic.SharedKernel.Analyzers" Version="0.1.0" />
```

## Documentation

For complete documentation including:

- Rule categories and IDs
- Configuration via `.editorconfig`
- Module flexibility options
- Code fix descriptions
